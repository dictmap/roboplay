## 提交策略

第一版只提交可阅读、可复现、可验证的轻量资料：

- Markdown 精讲
- Jupyter notebook
- 来源清单和验证 JSON
- 复现摘要与小规模结果

暂不提交远端原始视频、大体积日志、缓存目录和运行输出目录；这些后续可以按需用 Git LFS 或 release artifact 管理。
